** Configuration Flow

1. *Early Init* (early-init.el)
   - Package system setup
   - Performance optimizations
   - UI tweaks before GUI loads

2. *Main Init* (init.el)
   - UTF-8 configuration
   - Load core configs (ming-*.el)
   - Load tool configs (*-setup.el)
   - Load language configs (lang/*.el)

3. *Private Configs* (private/*.el)
   - Loaded automatically by setup files
   - Contains your personal settings
   - Git-ignored for privacy

** Data Storage Locations

Following XDG Base Directory specification:

| Type | Location | Description |
|------+----------+-------------|
| Mail data | =~/.local/share/mail/= | Email storage (Maildir) |
| Mail cache | =~/.cache/mu/= | mu search database |
| RSS data | =~/.emacs.d/elfeed/= | Feed database |
| Credentials | =~/.authinfo.gpg= | Encrypted passwords |
| Config | =~/.emacs.d/= | Emacs configuration |

* Setup Workflow

** New Installation

1. *Clone repository*
   #+BEGIN_SRC bash
   git clone <your-repo> ~/.emacs.d
   #+END_SRC

2. *Start Emacs*
   - Packages will auto-install

3. *Configure private settings*
   #+BEGIN_SRC bash
   cd ~/.emacs.d/private
   cp mail-private.example.el mail-private.el
   cp elfeed-private.example.el elfeed-private.el
   cp erc-private.example.el erc-private.el
   # Edit each file with your settings
   #+END_SRC

4. *Set up authentication*
   - Follow [[file:authinfo.org][Authentication Setup Guide]]
   - Create =~/.authinfo.gpg=

5. *Configure email (optional)*
   - Follow [[file:mail.org][Email Configuration Guide]]
   - Install mu and isync
   - Configure =~/.mbsyncrc=

6. *Restart Emacs*

** Adding New Tools

1. *Create setup file*
   - Add to =~/.emacs.d/lisp/tool-setup.el=

2. *Add require statement*
   - Add =(require 'tool-setup)= to =init.el=

3. *Create documentation*
   - Add =~/.emacs.d/docs/tool.org=
   - Update this INDEX.org

4. *Private configuration (if needed)*
   - Create =~/.emacs.d/private/tool-private.example.el=
   - Users copy to =tool-private.el=

* Troubleshooting

** General Issues

*** Packages Not Loading
#+BEGIN_SRC elisp
M-x package-refresh-contents
M-x package-install-selected-packages
#+END_SRC

*** Configuration Errors
#+BEGIN_SRC bash
emacs --debug-init
#+END_SRC

*** Check Messages
- =*Messages*= buffer shows errors and warnings
- =*Warnings*= buffer shows startup warnings

** Performance Issues

1. *Slow startup*
   - Check early-init.el optimizations
   - Review package list
   - Consider lazy loading

2. *High memory usage*
   - Close unused buffers
   - Restart Emacs periodically
   - Check for memory leaks in packages

* Contributing

** Documentation Style

All documentation should:
- Use Org format (=.org= files)
- Include clear table of contents
- Provide working examples
- Follow KISS principle
- Include troubleshooting section
- Cross-reference related docs

** File Naming

- Main docs: =topic.org= (e.g., =mail.org=, =irc.org=)
- Support files: =topic-appendix.org=
- All lowercase, hyphen-separated

** Organization

1. Place in =~/.emacs.d/docs/=
2. Update this INDEX.org
3. Link from README.org if major feature
4. Keep examples realistic and tested

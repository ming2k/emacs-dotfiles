* Common Tasks

** Setup New Tool

1. Create =~/.emacs.d/lisp/tool-setup.el=
2. Add =(require 'tool-setup)= to init.el
3. Create =~/.emacs.d/docs/tool.org=
4. Update =~/.emacs.d/docs/INDEX.org=
5. If private config needed: create =private/tool-private.example.el=

** Update Mail

Terminal:
#+BEGIN_SRC bash
mbsync -a && mu index
#+END_SRC

In Emacs:
Press =U= in mu4e

** Update RSS Feeds

Press =G= in Elfeed (C-c e to open)

** Check Configuration Errors

#+BEGIN_SRC bash
emacs --debug-init
#+END_SRC

Or check =*Messages*= buffer in Emacs

** Reload Configuration

#+BEGIN_SRC emacs-lisp
M-x eval-expression RET (load-file user-init-file) RET
#+END_SRC

Note: early-init.el requires full restart

* Search Queries

** Email Search (mu4e)
| Query                    | Description                |
|--------------------------+----------------------------|
| from:john                | From specific sender       |
| to:me@example.com        | To specific recipient      |
| subject:meeting          | Subject contains           |
| date:7d..now             | Last 7 days                |
| flag:unread              | Unread messages            |
| flag:attach              | Has attachments            |
| size:1M..                | Larger than 1MB            |

** RSS Search (Elfeed)
| Filter               | Description                |
|----------------------+----------------------------|
| +unread              | Unread entries             |
| +emacs               | Tagged "emacs"             |
| @1-week-ago          | From last week             |
| +emacs +news         | Tagged both "emacs" & "news" |
| @1-week-ago +unread  | Unread from last week      |

* Troubleshooting Quick Fixes

** Packages Not Loading
#+BEGIN_SRC emacs-lisp
M-x package-refresh-contents
M-x package-install-selected-packages
#+END_SRC

** Email Not Syncing
#+BEGIN_SRC bash
mbsync -V  # Verbose mode to see errors
mu index --rebuild
#+END_SRC

** Authentication Failing
#+BEGIN_SRC bash
# Test decryption
gpg -d ~/.authinfo.gpg

# Check format
cat ~/.authinfo  # If using plain text
#+END_SRC

** GPG Issues
#+BEGIN_SRC bash
# Restart GPG agent
gpgconf --kill gpg-agent
gpg-agent --daemon

# Test
gpg -d ~/.authinfo.gpg
#+END_SRC
